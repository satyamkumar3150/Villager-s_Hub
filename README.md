# 🍔 Villagers Hub

Villagers Hub is a simple food delivery website inspired by platforms like Grubhub. I built this project to understand how real-world food ordering systems work and to improve my frontend development skills.

##  What this project does
 
- Simulates placing an order  
- Responsive design (works on mobile and desktop)  

##  Why I made this

I wanted to build something practical instead of just small practice projects. Food delivery apps are widely used, so I chose this idea to learn how such platforms are designed and structured.

## Tech Used

- HTML  
- CSS  
- JavaScript  

## What I learned

While working on this project, I researched how real-world applications are built. I learned that a complete system includes frontend, backend, databases, and cloud services working together.

Even though this project is frontend-only, it helped me understand how these different parts connect in an actual product.
